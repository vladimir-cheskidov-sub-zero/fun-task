# Prompt

Создай гидратор доменной модели Category.

Ограничения:
- Источник данных для гидратора `data/categories.json`.
- Интерфейс гидратора должен находиться в слое Domain.
- Реализация гидратора должна находиться в слое Infrastructure.
- Гидратор может бросать только доменные исключения, так как реализует доменный интерфейс.
- Провести обработку ошибок.
