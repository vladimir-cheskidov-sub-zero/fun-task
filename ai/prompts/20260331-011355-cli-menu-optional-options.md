- cli-menu-optional-options
Исправь параметры CLI команды
* adult
* region
* staff
так как они опциональны.
Для исправления параметров типа enum использовать null object (default behaviour) шаблон проектирования, чтобы избежать nullable значений там где передаются enums. Для этого просто создай в перечислениях пункты `unspecified`, что будет означать, что значение не передано и требуется default поведение.
